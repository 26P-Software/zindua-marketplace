# Zindua Marketplace: React Learning Path

Welcome to the **Zindua Marketplace** repository! This project serves as the central hub for the React module of our curriculum. Here, we provide structured lessons, guidance, and hands-on projects to take you from a React beginner to a confident developer.

## Important: Where to find the notes

All lesson notes, code samples, and weekly guides are organized within the `src/slides/` folder. Please navigate there to access the detailed Markdown files for each week.

## Purpose

This repository is designed to accompany our React course. It contains the core concepts, code snippets, and project guidelines broken down into a 4-week intensive structure to help you build real-world applications.

## Course Curriculum Breakdown

### Week 1: React Fundamentals

* **Lesson 1:** Introduction to React, SPAs, and Environment Setup.
* **Lesson 2:** JSX and Functional Components.
* **Lesson 3:** State (`useState`) and Event Handling.
* **Lesson 4:** Conditional Rendering and List Management (`.map`).

### Week 2: Intermediate React

* **Lesson 1:** Forms, Controlled Components, and `useEffect`.
* **Lesson 2:** Modern UI Development with Tailwind CSS and `shadcn/ui`.
* **Lesson 3:** State Management with the Context API.
* **Lesson 4:** Multi-page Routing with `react-router-dom`.

### Week 3: Advanced React

* **Lesson 1:** Backend Integration (API Data Fetching).
* **Lesson 2:** Performance Optimization (`memo`, `useMemo`, `useCallback`).
* **Lesson 3 & 4:** Testing with Jest and React Testing Library.

### Week 4: Real-World Application

* **Lesson 1:** State Management with Redux Toolkit.
* **Lesson 2:** Implementing CRUD Functionality.
* **Lesson 3:** Deployment Best Practices and Production Builds.

## Getting Started

Each week's notes and challenges can be found in the `src/slides/` folder.

1. **Clone the repository:**
```bash
git clone https://github.com/your-username/zindua-marketplace.git

```


2. **Setup your environment:**
Ensure you have [Node.js](https://nodejs.org/) installed.
3. **Follow the guide:**
Open the files in `src/slides/` and follow the daily challenges to reinforce your learning.
